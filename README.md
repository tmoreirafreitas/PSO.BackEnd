# PSO.BackEnd
Portal de Serviço de Ótica API é um sistema em RestFull criado em WebApi Core para gerenciamento de serviços de óticas.
A estrutura do sistema foi baseada em CQRS.

Tecnologias utilizadas:
* .NET CORE 2.2
* WebAPI
* MediatR
* PostgreSQL
* MongoDB
* AutoMapper
* EntityFramework
* Swagger
* Docker
